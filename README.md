
# THREADS_CLONE

Feature List:

🌟 Tech stack: NODEJS + EXPRESS JS 
🎃 Authentication & Authorization with JWT
📝 Create Post
🗑️ Delete Post
❤️ Like/Unlike Post
💬 Comment to a Post
👥 Follow/Unfollow Users
❄️ Freeze Your Account


## Setup .env file

```bash
PORT=...
MONGO_URI=...
JWT_SECRET=...
CLOUDINARY_CLOUD_NAME=...
CLOUDINARY_CLOUD_KEY=...
CLOUDINARY_API_SECRET=...
```

## Install dependencie

```bash
  npm Install
```


## Build the app


```bash
  npm run build
```

## Start the app

```bash
  npm start
```
